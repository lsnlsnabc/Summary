Python基础

Python创建list
Python内置一种数据类型是列表：list。
  • list是一种有序集合，可以随时添加和删除其中的元素。
  • list为有序集合，其中元素按照顺序排列。
  • 由于Python是动态语言，所以list中包含的元素并不要求都是同一种数据类型
L = {'Adam':95.5, 'Lisa':85, 'Bart':59}
a=sorted(L.iteritems(), key=lambda d:d[1],reverse=True)
print a

print L[0]  #索引超出范围报错

print L[-1] #负数索引代表倒序，即倒数第一个数

L = {'Adam','Lisa','Bart'}
L.append('Paul') #末尾添加一个元素
L.insert(0,'Paul') #在指定位置添加元素
L.pop() #删除list中的最后一个元素
L.pop(2) #删除指定位置的元素
L[2] = 'Paul' #替换元素

Python创建tuple
tuple是另一种有序列表，一旦创建完毕就不能修改了
t = ('Adam','Lisa','Bart')
  • 因为()既可以表示tuple，又可以作为括号运算时的优先级，结果(1)被Python解释为整数1，单元素tuple要多家一个逗号“,”，避免歧义
t = (1,)
  • tuple的不变是说tuple的每个元素指向永远不变

Python之while循环
  • while循环语句用法

print sum

Python之dict
  • dict是通过key来查找value
d={
'Adam':95,
'Lisa':85,
'Bart':59
}
print d['Adam']
  • 避免发生KeyError有两个办法
a)先判断一下key是否存在，用in操作符
if 'Paul' in d:
    print d['Paul']
b)dict本身提供一个get方法，在Key不存在时，返回None
print d.get('Paul')
