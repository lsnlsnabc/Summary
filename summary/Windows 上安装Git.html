Windows 上安装Git

安装Git
在Windows上使用Git，可以冲Git官网直接下载安装程序https://git-scm.com/downloads
打开Git Bash，设置命名和邮箱
猫猫&king@DESKTOP-J13LTJN MINGW64 ~
$ git config --global user.name "lsnlsnabc"
猫猫&king@DESKTOP-J13LTJN MINGW64 ~
$ git config --global user.email "lsnlsn_abc@163.com"

创建版本库
首先在合适的地方创建空目录
猫猫&king@DESKTOP-J13LTJN MINGW64 ~
$ cd D:
猫猫&king@DESKTOP-J13LTJN MINGW64 /d
$ mkdir gitStore
猫猫&king@DESKTOP-J13LTJN MINGW64 /d
$ cd gitStore
猫猫&king@DESKTOP-J13LTJN MINGW64 /d/gitStore
$ pwd    //查看当前目录
/d/gitStore
通过git init 命令把这个目录变成Git可以管理的仓库，生成一个隐藏的.git目录
猫猫&king@DESKTOP-J13LTJN MINGW64 /d/gitStore
$ git init
Initialized empty Git repository in D:/gitStore/.git/
猫猫&king@DESKTOP-J13LTJN MINGW64 /d/gitStore (master)
$ ls -ah
./  ../  .git/

在github上 your profile中的Repositories中创建一个仓库
在git Bash中运用如下命令将文件上传至仓库中
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/lsnlsnabc/TCP.git
git push -u origin master
$ git remote rm origin
